1. Request for comment process

    Fall
    2019

    We'll launch a Rivet Request For Comment (RFC) process, which we'll use as a place to discuss and solicit feedback on breaking changes and major new features.

  2. Rivet 2.0-alpha development

    Winter
    2019

    We'll create a new dedicated v2 development branch, begin adding new components, and make an initial alpha release of the Rivet 2.0 package. We'll also release a detailed end-of-life plan for version 1.x.

  3. Rivet site redesign

    Spring
    2020

    At the start of 2020 we'll begin work on a new Rivet website that will include documentation for brand fundamentals (type, color, logos, voice and tone), page templates, and new components.

  4. 2.0 Launch

    Summer
    2020

    In the Spring we'll start the transition process to the new documentation site becoming the default at rivet.iu.edu and release version 2.0.0.
